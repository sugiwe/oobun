# 機能仕様

## 画面構成

### トップページ (`/`)

- 購読中の Thread 一覧（ログイン時）
- 購読がない場合は「おすすめ／全体公開 Thread」一覧
- 各 Thread は最新1〜3件の Post をプレビュー表示

---

### Thread 詳細ページ (`/:slug`)

- Thread の概要
- 参加ユーザー情報(書き手)
- Post 一覧（抜粋表示: 300文字程度）
- 各 Post に「続きを読む」リンク → Post 詳細ページへ
- ページネーション(`?page=2`)で過去ログ閲覧
- 購読ボタン(「購読する」/「購読中」)
- RSS フィードリンク(個別スレッド単位)

---

### Post 詳細ページ (`/:slug/:post_id`)

- Post の全文表示
- 投稿者情報
- 投稿日時
- 前後の Post へのナビゲーション(「← 前の投稿」「次の投稿 →」)
- Thread に戻るリンク
- OGP 対応(将来)

---

### ユーザーページ (`/@username`)

- プロフィール
- 参加中の Thread 一覧(書き手として)
- 購読中の Thread 一覧

---

## コアコンセプト

- オープン（公開）が基本
- スレッド単位の文通
- 投稿は基本的に交代制
- 購読(Subscription)の概念あり
- 将来的にクローズド／有料購読の余地を残す

リアルタイムチャットではなく、蓄積されていく対話を「読み物」として楽しむことに価値を置きます。
